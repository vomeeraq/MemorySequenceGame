ИГРА "Запомни последовательность" (Memory Sequence Game)

Описание:
Настольная игра на Java Swing для тренировки памяти и внимания. Вам предстоит запоминать и повторять усложняющиеся последовательности цветов.

Особенности:
Классический игровой процесс: запоминайте и воспроизводите последовательности из 4 цветов.
Гибкие настройки: регулируйте количество игроков (от 1 до 5), число жизней и скорость демонстрации.
Функция "Подсказка": поможет вспомнить последний элемент последовательности.
Результаты игры: показывает победителя и статистику по жизням игроков.
Как запустить игру:

1. Требования к системе: Java Runtime Environment (JRE)
Для работы игры "Запомни последовательность" на вашем компьютере обязательно должна быть установлена среда выполнения Java (JRE) версии 1.8.0 (Java 8) или новее.
Что такое JRE? JRE (Java Runtime Environment) – это необходимый программный компонент, который позволяет операционной системе вашего компьютера запускать приложения, написанные на языке Java. Без JRE игра не сможет запуститься, и вы можете увидеть сообщение об ошибке.

Как установить JRE:
Скачайте JRE: Перейдите на официальную страницу загрузки Java от Oracle: https://www.oracle.com/java/technologies/downloads/. На этой странице найдите раздел "JRE" или "Java Runtime Environment" и выберите версию, совместимую с вашей операционной системой (Windows, macOS, Linux).
Запустите установщик: Найдите скачанный файл (например, с расширением .exe для Windows или .dmg для macOS) и запустите его. Следуйте всем инструкциям, которые появятся на экране мастера установки.
Перезагрузите компьютер: После успешного завершения установки рекомендуется перезагрузить ваш компьютер. Это гарантирует, что все системные изменения будут применены корректно.
Возникли трудности? Если после установки JRE у вас все равно возникают проблемы с запуском игры, пожалуйста, обратитесь к полной и подробной инструкции, расположенной непосредственно в репозитории проекта на GitHub. Она содержит более детальные шаги, советы по устранению распространенных неполадок и ответы на часто задаваемые вопросы.

2. Запуск файла игры:

Для Windows:
Скачайте файл Запомни последовательность.exe из папки Игра/ в репозитории.
Дважды щелкните по скачанному файлу Запомни последовательность.exe. Игра должна запуститься.

Для других систем (macOS, Linux и Windows, если .exe не работает):
Скачайте файл Запомни последовательность.jar из папки Игра/ в репозитории.
Дважды щелкните по скачанному файлу Запомни последовательность.jar. Игра должна запуститься. Если этого не происходит, убедитесь, что JRE установлена корректно и ассоциирована с файлами .jar.

Структура репозитория:
Игра/: Скомпилированные исполняемые версии игры (.jar, .exe).
Коды/: Весь исходный код Java (.java файлы).
README.txt: Этот файл с описанием проекта и инструкциями.

Использованные технологии:
Java SE

Требования:
Для запуска игры необходима установленная Java Runtime Environment (JRE) версии 1.8.0 (Java 8) или новее.
